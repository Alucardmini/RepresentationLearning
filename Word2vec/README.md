# Word2vec

1. Word2Vec模型
	
    word2vec.c: https://code.google.com/archive/p/word2vec 

    distance.c: 基于cosine相似度找词向量的近邻
	
    binary_translation.c: 将二进制词向量转化为文本向量

2. Word2vec+TransE模型
	
    word2vec_transE.c: 在训练Word2vec时，加入三元组信息，使得词向量训练更加充分
    
    stopwords.txt: 停用词
